# Strategic Briefing: Wellness Social Network (Mobile + Web)

## 1. Product Vision
An exclusive platform connecting users with high-quality private clubs. The aesthetic moves away from "dark clichés" towards botanical luxury, health, and sophistication.
*   **Inspiration:** Airbnb (maps/reservations) and Headspace (visual peace).

## 2. User Stories (Navigation Flows)
### For the End User (Mobile App)
*   **Discovery:** See a map with custom pins to find nearby clubs.
*   **Decision:** View high-quality photos and the variety menu (THC/CBD) before visiting.
*   **Loyalty:** Save favorite clubs to a private list for quick menu access.
*   **Social:** Leave reviews with photos to help the community.

### For the Club (B2B Web Panel)
*   **Image Management:** Upload facility photos to attract premium users.
*   **Inventory Control:** Instantly update prices or mark items as "Out of Stock" via mobile/tablet.
*   **Reputation:** View and respond to customer reviews.

## 3. Information Architecture (Sitemap)
### Mobile App (4-Icon Tab Bar)
1.  **Explore (Home):** Search bar, quick filters (Sativa, Indica, CBD, Terrace, Coworking), vertical feed cards.
2.  **Map:** Full-screen map (Dark or Retro mode) with custom club logo markers.
3.  **Favorites:** Simplified list of saved clubs.
4.  **My Profile:** Settings, review history, membership verification.

### Web Panel (Sidebar Menu)
1.  **Dashboard:** Quick stats (profile visits, low stock).
2.  **My Club:** Profile editor (photos, description, hours, coordinates).
3.  **Inventory:** Interactive table for varieties (Name, % THC, Category, Price).
4.  **Reviews:** List of user comments.

## 4. Visual Requirements (UI Kit)
*   **Typography:** Serif for titles (elegance), Sans-serif for body (legibility).
*   **Palette:** Forest Green (#2D463E), Cream (#F5F5F1), Gold/Soft Black accents.
*   **Components:**
    *   **Badges:** Color-coded (e.g., Soft green for Sativa, Pastel purple for Indica).
    *   **Cards:** Rounded corners, soft shadows, high-res photography.
    *   **Inputs:** Clean forms with clear labels and elegant error feedback.
*   **Mode:** Primary Light Mode; Dark Mode uses deep grays instead of pure black.

## 5. Design Prompt Summary
Minimalist mobile and web administrative interface for a premium botanical club network. Aesthetic: "Luxe & Wellness". High visual hierarchy for local photos and clarity for THC/CBD information.
